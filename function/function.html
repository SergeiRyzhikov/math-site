Математические операции

  • Умножение: * Пример(5*x)
  • Деление: / Пример(5.5/x)
  • Возведение в степень: ^ Пример(x^3)
  • Плюс: + Минус: -
  • Для ввода дробных чисел используется точка. Пример(4.4)
  • Порядок выполениния действий указывается спомщью скобок. Пример((5/x)-(6-x*5))

Также возможен ввод тригонометрических фуннкций

  • Синус - sin(x)
  • Косинус - cos(x)
  • Тангенс - tg(x)
  • Котангенс - ctg(x)

Константы

  • Число Пи: pi
  • Число Эйлера: e
